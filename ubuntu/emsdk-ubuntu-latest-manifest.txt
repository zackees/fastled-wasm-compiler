Reconstruction Instructions:
1. Download all emsdk-ubuntu-latest.tar.xz.part* files to the same directory
2. Download emsdk-ubuntu-latest-reconstruct.sh to the same directory
3. Run: chmod +x emsdk-ubuntu-latest-reconstruct.sh
4. Run: ./emsdk-ubuntu-latest-reconstruct.sh
5. Extract: tar -xJf emsdk-ubuntu-latest.tar.xz
6. Setup: cd emsdk && source ./emsdk_env.sh

Alternative manual reconstruction:
1. Download all parts to the same directory  
2. Run: cat emsdk-ubuntu-latest.tar.xz.part* > emsdk-ubuntu-latest.tar.xz
3. Extract: tar -xJf emsdk-ubuntu-latest.tar.xz
4. Setup: cd emsdk && source ./emsdk_env.sh
