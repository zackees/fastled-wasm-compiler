Reconstruction Instructions:
1. Download all emsdk-windows-latest.tar.xz.part* files to the same directory
2. Download emsdk-windows-latest-reconstruct.sh to the same directory
3. Run: chmod +x emsdk-windows-latest-reconstruct.sh
4. Run: ./emsdk-windows-latest-reconstruct.sh
5. Extract: tar -xJf emsdk-windows-latest.tar.xz
6. Setup: cd emsdk && source ./emsdk_env.sh

Alternative manual reconstruction:
1. Download all parts to the same directory  
2. Run: cat emsdk-windows-latest.tar.xz.part* > emsdk-windows-latest.tar.xz
3. Extract: tar -xJf emsdk-windows-latest.tar.xz
4. Setup: cd emsdk && source ./emsdk_env.sh
